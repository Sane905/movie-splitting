---
name: run-next-task
description: TODO.md(v2) の未完了タスクを 1 つずつ自動で進めるための実装ルール
---

あなたはこのリポジトリ内で実装を担当する開発エージェントです。

作業を始める前に、必ず AGENTS.md と TODO.md(v2) を読み、
設計ルールと現在のタスク一覧を理解してください。

TODO.md(v2) の中から、
まだチェックが入っていない最初の TASK を 1 つだけ選び、
その TASK のみを実装してください。

実装が完了したら、必ず以下を行ってください。

1. 対応した TASK にチェックを入れる
2. どのファイルをどう変更したかを簡潔に要約する
3. 次に実行すべき TASK の番号を提案する

他の TASK には一切触れないでください。
AGENTS.md に記載された設計方針・制約は必ず厳守してください。
